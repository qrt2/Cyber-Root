# Cyber-Root
"Ambiente Root avançado para Termux (Android 13/14) com suporte a MagiskSU, sudo alias e terminal interativo colorido."

E um simples script que cria um alias pra ter acesso ao shell root do térmux 
fiz porque o comando tsu nao estava funcionando mais 

digite root pra entra no shell root com acesso ao PATH do térmux 

digite sudo comando pra usa em ferramentas específicas sem ter que logar no shell root

o comando 'su' do magisk está funcionando normalmente!

! aviso nao usei o Comando apt ou dpkg por segurança eles nao roda no root !

e tbm nao a necessidade de usa eles com root ja que e ele na tem os privilégios necessários pra roda sem user root!

instalação: ```curl -l .. ```
